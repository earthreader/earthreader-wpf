ID			: 해당 아이템의 ID, 프로그램 실행 시마다 변경될 수 있음. 프로그램 실행 중에는 고정. RootID는 0
ParentID	: 부모의 ID를 저장. RootID의 ParentID는 0
IsFeed		: 해당 아이템이 피드일 경우 true, 아니면 false
URL			: 해당 아이템이 피드일 경우 피드의 URL
Childred	: 해당 아이템이 카테고리일 경우, 자식들의 ID를 나열한 리스트
Caption		: 아이템의 제목. 카테고리일 경우 변경 가능
Count		: 새로운 피드의 갯수를 저장
Opacity		: 어디에 쓰는거였더라
